# MyTasks
### Учебный проект
 - каждый пользователь может создавать задачи
 - админ может изменять 
 
 ### Требования
 - PHP >= 7.1
 - MySQL >= 5.6 
 
 ### Установка:
 - composer install
 - Создать пустую БД
 - внести реквизиты доступа к БД в файл config.php
 - Выполнить SQL миграции из файла schema.sql
